## Project Scope (MVP)

| Feature | Description |
|---------|-------------|
| Consent Engine | Capture, store, validate, revoke with digital signatures |
| Dynamics Integration | Real-time bi-directional sync with Dynamics 365 |
| HIPAA Compliance | Pre-built templates, immutable audit trail, authorization workflows |
| Admin Dashboard | Consent overview, compliance metrics, user management |
| Patient Portal | Self-service preferences, consent history, revocation |
| Full API | REST API for Power Automate, Zapier, n8n integrations |
